To classify an image :

1) Specify the image path in 'in.m' code.

2) Run 'in.m' in Matlab and copy pixel data from 'inputImage' table into Excel Sheet for HH-,HV-,VV- bands.

3) Copy data from Excel Sheets into Text files for the data sets.

4) Run the code "sampleread.c" in Code Blocks.

5) See the percentage labelling as output window.
